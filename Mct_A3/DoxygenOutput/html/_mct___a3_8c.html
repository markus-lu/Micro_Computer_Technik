Funktion zum Speichern der Multiplikanten und Berechnen des Produkts

Parameter
keysHier werden die Taster und Schalter übergeben. (0 - 0x7F)
AHier wird der erste Multiplikant gespeichert, sobald der Taster gedrueckt wird. (0 - 0xF)
BHier wird der zweite Multiplikant gespeichert, sobald der Taster gedrueckt wird. (0 - 0xF)
CHier wird das Produkt gespeichert, sobald beide Multiplikanten gespeichert wurden. (0 - 0xFF)
valuesAvailableHier wird eine 2-Bit Flagge gespeichert, die anzeigt, welche Multiplikanten schon gespeichert wurden. (Bit 0 gesetzt: Multiplikant A gespeichert, Bit 1 gesetzt: Multiplikant B gespeichert)
